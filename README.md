# ATM Management System

A Java console-based ATM simulation that allows users to manage bank accounts and perform basic banking operations.

## Features

* Create account
* Check account balance
* Deposit money
* Withdraw money
* Exit system

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList (Collections Framework)

## How to Run

1. Open the project in Eclipse or any Java IDE.
2. Run `Main.java`.
3. Use the console menu to perform ATM operations.
